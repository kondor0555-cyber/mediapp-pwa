# 📱 MediApp – Υπενθυμίσεις Φαρμάκων, Ραντεβού & Μετρήσεων by George Giannopoulos (zawasoft) 
**Έξυπνη καθημερινή οργάνωση υγείας – σε μορφή PWA εφαρμογής**

Το MediApp είναι μία ελαφριά web εφαρμογή που λειτουργεί και σαν **Native App** μέσω τεχνολογίας **PWA (Progressive Web App)**.  
Δεν απαιτεί εγκατάσταση από Play Store ή κατάστημα – γίνεται εγκατάσταση απευθείας από τον browser.

---

## 🚀 Χαρακτηριστικά

- ✔ **Υπενθυμίσεις φαρμάκων** με καθημερινή επιλογή  
- ✔ **Υπενθυμίσεις ραντεβού** με ειδοποίηση 3 ώρες πριν  
- ✔ **Μετρήσεις υγείας**
  - Παλμοί  
  - Ζάκχαρο  
  - Πίεση  
  - Οξυγόνο  
- ✔ **Αποθήκευση δεδομένων τοπικά** (δεν χρειάζεται internet)  
- ✔ **Εκτυπώσεις** κάθε κατηγορίας μετρήσεων  
- ✔ **Offline λειτουργία** (service worker)  
- ✔ **Ειδοποιήσεις συστήματος** σε PC & κινητά  
- ✔ **Ελληνικά & Αγγλικά**  
- ✔ Ελαφρύ, γρήγορο, χωρίς εγκατάσταση προγραμμάτων

## 📥 Πώς το εγκαθιστά ο χρήστης

### 🖥️ **1. Σε Υπολογιστή (Chrome ή Edge)**

1. Ανοίξτε το link της εφαρμογής:  
   👉 **https://\<username\>.github.io/mediapp-pwa/**
2. Στη γραμμή διεύθυνσης θα εμφανιστεί κουμπί:  
   **“Εγκατάσταση εφαρμογής” / “Install App”**
3. Πατήστε το → δημιουργείται κανονική εφαρμογή στα Windows.
4. Μπορεί να προστεθεί στην εκκίνηση του συστήματος αν μπει στον φάκελο `shell:startup`.
---------------------------------------------------------------------------------
Οδηγίες για startup: 

1 Άνοιξε Start → βρες το MediApp (θα φαίνεται σαν app).

2 Κάνε δεξί κλικ → Open file location (ή "Άνοιγμα τοποθεσίας αρχείου").

3 Αντιγραφή του shortcut.

4 Win + R → γράψε shell:startup → Enter.

5 Κάνε επικόλληση (Paste) το shortcut στον φάκελο Startup.

Από εδώ και πέρα:

Όταν ανοίγεις Windows & μπαίνεις στον λογαριασμό σου, θα ανοίγει αυτόματα το MediApp PWA.

---------------------------------------------------------------------------------

### 📱 **2. Σε Android (Chrome)**

1. Ανοίξτε το ίδιο link.  
2. Ο browser θα προτείνει:  
   **“Προσθήκη στην Αρχική Οθόνη” / “Add to Home Screen”**
3. Πατήστε το → εμφανίζεται σαν κανονική εφαρμογή στη συσκευή σας.
